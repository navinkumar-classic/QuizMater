# QuizMaster-V1
This is a project to build a quizzing website for my MAD-1 project
```bash
pip install Flask
pip install Flask-SQLAlchemy
python app.py
```

Open [http://127.0.0.1:5000](http://127.0.0.1:5000) with your browser to see the result.
